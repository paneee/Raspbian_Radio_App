
# raspbian_radio_app

raspbian_radio_app is a a Flutter for android or ios phone application 


## Two pagess manage the application
* ### Page 1 - radio management:
  * #### Change speaker volume
  * #### Radio selection
  * #### Switching the radio play and stop
  * #### Change radio volume
